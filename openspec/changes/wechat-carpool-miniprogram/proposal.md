## Why

随着共享经济的发展和环保出行理念的普及，拼车出行成为越来越多人的选择。当前市场缺乏一个轻量级、易用的拼车信息发布和匹配平台，用户需要一个便捷的工具来发布和查找拼车信息，同时能够直接与对方沟通协商。本小程序旨在解决这一痛点，提供一个基于微信生态的拼车信息发布和匹配平台。

## What Changes

创建一个全新的微信拼车小程序，包含以下核心功能：

- **双向拼车发布**：支持"人找车"和"车找人"两种发布模式
- **智能匹配推荐**：基于地理位置和路线的智能匹配算法
- **即时通讯系统**：内置 IM 功能，方便车主和乘客实时沟通
- **用户认证体系**：实名认证和信用评价系统
- **行程管理**：完整的行程状态跟踪和历史记录
- **地图可视化**：集成地图展示路线和上车点
- **个人中心**：拼车记录、统计数据、个人设置等

技术栈：
- 前端框架：uni-app + Vue3
- UI 组件库：uView UI
- 地图服务：腾讯地图/高德地图
- IM 服务：腾讯云 IM SDK / 自建 WebSocket

## Capabilities

### New Capabilities

- `ride-publishing`：拼车信息发布能力，支持人找车和车找人两种模式
- `ride-matching`：智能匹配推荐能力，基于地理位置和路线匹配合适的拼车信息
- `instant-messaging`：即时通讯能力，支持车主和乘客实时对话
- `user-authentication`：用户认证能力，包括实名认证和身份验证
- `ride-management`：行程管理能力，包括行程状态跟踪、历史记录和取消
- `rating-system`：评价系统能力，支持车主和乘客互评及信用积分
- `map-visualization`：地图可视化能力，展示路线、上车点和实时位置
- `user-profile`：个人中心能力，包括个人信息、拼车记录和统计数据
- `notification-system`：消息通知能力，支持系统通知和消息推送
