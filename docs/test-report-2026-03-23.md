## 修复内容总结

本次修复了微信拼车小程序的4个关键问题:

### 1. 首页下拉刷新卡住 ✅
**提交:** edd9efb
**修复内容:**
- 添加 `onPullDownRefresh` 生命周期函数
- 在 `finally` 块中调用 `uni.stopPullDownRefresh()` 停止刷新动画
- 重置分页状态并重新加载列表

### 2. 首页联系TA聊天跳转白屏 ✅
**状态:** 已在之前的提交中修复
**当前代码:** 跳转路径为 `/pages/chat/detail`,正确

### 3. 首页 onShow 自动刷新 ✅
**提交:** d2103ea, 28760e0
**修复内容:**
- 添加 `onShow` 生命周期函数
- 使用 `lastLoadTime` 时间戳机制(5秒阈值)
- 在 `onMounted` 中初始化时间戳
- 搜索状态下不自动刷新,避免丢失搜索结果

### 4. 发布页表单重置不完整 ✅
**提交:** df33357
**修复内容:**
- 在 `resetForm()` 中添加日期时间重置逻辑
- 将 `selectedDate` 和 `selectedTime` 重置为当前时间
- 调用 `updateDepartureTime()` 同步到 `formData.departureTime`

### 5. 后端 API 未返回坐标数据 ✅
**提交:** c630937
**修复内容:**
- 在 `getRideById` 方法中添加 `departureLocation`, `destinationLocation`, `waypoints` 字段
- 在 `findRides` 方法中添加相同的坐标字段
- 确保前端可以正确渲染地图路线
